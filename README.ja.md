# irodori-docs

Irodori Tableの公開ドキュメントサイト。

オンラインで読む：

<https://hjosugi.github.io/irodori-docs/>

## 開発

```sh
cargo install mdbook
mdbook serve
mdbook build
```

Markdownのソースは`src/`にあります。生成されたリファレンスページは`irodori-table`からミラーされています。

ライセンス: `MIT OR 0BSD`。

## ライセンス

0BSD。ほぼすべての目的でこのプロジェクトを使用、コピー、修正、配布できます。
