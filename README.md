# irodori-docs

Public documentation site for Irodori Table.

Read it online:

<https://hjosugi.github.io/irodori-docs/>

## Develop

```sh
cargo install mdbook
mdbook serve
mdbook build
```

Markdown source lives in `src/`. Generated reference pages are mirrored from
`irodori-table`.
